# LTHDT-GameTetrisOOP
### Hướng dẫn chơi
1. Nhấn phím A, D để điều khiển hình sang trái, phải
2. Nhấn phím Enter để xoay hình
3. Nhấn phím +, - để tăng, giảm tốc độ rơi
